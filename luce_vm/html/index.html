Welcome to LuceVM

It's working! Congratulations on setting up LuceVM™

LuceVM is a self-contained virtual machine to facilitate web3 development using the python stack.
Most importantly LuceVM allows to compile, deploy and interact with Ethereum Smart Contracts
from a Python environment in a seamless manner. It also provides additional functionality but let's
keep it simple for now.
